# AGENTS.md

This project uses `CLAUDE.md` as the single source of truth for all AI agent instructions.

**For any AI coding assistant:** Read [CLAUDE.md](./CLAUDE.md) for project context, architecture, conventions, and prohibitions.

**For Claude Code users:** The `.claude/` directory contains additional capabilities — skills (phased workflows), rules (code patterns), and protocols (behavioral rules). These require Claude Code's native skill system.

**For other tools:** Your project context, architecture overview, coding conventions, and absolute prohibitions in CLAUDE.md work with any AI that reads markdown. Tool-specific features (skills, rules, protocols) can be adapted to your tool's native format — see [PROVIDERS.md](./PROVIDERS.md) for documentation links.
